*** Field Notes

A React Native app for cheking a gps location and making notes

Native function --- Uses `expo-location` to get the user's GPS coordinates and `reverseGeocodeAsync` to find the address.
Api ---- Uses `axios` to fetch a random quote from `dummyjson.com/quotes/random` to fill the description.
Maps --- The details screen can open the location in the phone's native maps app using `Linking`.


This is a Bare workflow app

1  Install NDK from the Android Studio SDK Manager.
2  Install dependencies:`npm install`
3  Launch an Android Emulator.
4  Run the app: `npm run android`
